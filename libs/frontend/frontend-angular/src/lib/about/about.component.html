Beschrijving van project

Ik ga een website maken waarop mensen motoren kunnen bekijken en kunnen reviewen. Een user kan reviews plaatsen op meerdere motors en alle reviews van andere users bekijken. Reviews worden geplaats op modellen, niet op unieke exemplaren van eigenaren. Dit is een grof voorbeeld van wat ik bedoel (scroll onderaan voor user reviews).

Naast het plaatsen van reviews, kunnen users ook de reviews van andere users beoordelen. Ze kunnen een 'like' of 'dislike' geven aan bestaande reviews. Hiermee kunnen users zien welke reviews het meest behulpzaam zijn, en kunnen reviews die veel likes hebben bovenaan de pagina komen zodat die meer prominent worden.

Users kunnen aangeven welke motor(s) zij zelf bezitten, hiermee kunnen users aangeven welke motors ze rijden en ook aangeven bij hun geplaatste reviews of zij de eigenaar zijn van de motoren die zij hebben reviewed. Users kunnen ook gangs aanmaken en aan gangs deelnemen. Een 'Gang' is een groep motorrijders die bijelkaar horen (e.g. Hell's Angels). Gangs kunnen events aanmaken. 'Events' zijn meestal 'get togethers' waarin er grote groepen motorrijders samen de weg op gaan. Users kunnen hunzelf inschrijven voor events om aan te geven dat ze zullen deelnemen. Wanneer een user zich inschrijft voor een event, moet hij ook aangeven met welke motor hij zal komen. Hij kan kiezen uit een van de motoren waarvan hij zelf de eigenaar is.

Op elke motor review pagina krijgt de gebruiker te zien welke motoren andere users ook leuk vinden naast de huidige motor. Hiermee krijgt de gebruiker aanbevelingen voor andere motors te zien die hij misschien, net als de huidige motor, interessant zou vinden.
In het geval dat een gebruiker een negatieve review plaatst bij een motor krijgt die gebruiker motoren te zien die positief zijn reviewed door users die net als de gebruiker bij de huidige motor een negatieve review hebben achtergelaten.
De pagina laat ook zien bij welke events de huidige motor veel staat ingeschreven. Hierdoor kan de user misschien interesse krijgen voor motor events die hij eerder niet had gezien. Ook laat de pagina zien bij welke gangs de motor veel in voor komt, waardoor de user misschien interesse krijgt om aan bepaalde gangs deel te nemen.

Het aanmaken van motoren mag alleen gedaan worden door gebruikers met de "Admin" rol. Wanneer een motor is toegevoegd kunnen alle users de nieuwe motor bekijken en reviews achterlaten.
